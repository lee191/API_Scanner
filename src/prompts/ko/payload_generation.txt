이 API 엔드포인트에 대한 보안 테스트 페이로드를 생성하세요.

엔드포인트: {method} {url}

다음 항목을 테스트할 페이로드를 생성하세요:
- SQL 인젝션
- XSS
- 인증 우회
- 명령 인젝션
- 경로 순회
- IDOR

다음 JSON 형식으로 반환하세요:
{{
    "payloads": [
        {{
            "type": "테스트_유형",
            "payload": "실제_페이로드_문자열",
            "description": "무엇을 테스트하는지"
        }}
    ]
}}
